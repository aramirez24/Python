# Pràctica 2 - Calculadora!

## Enunciat de la pràctica

Fes una calculadora interactiva amb Python. El programa ha de demanar les dades necessàries (l’ordre pot ser el que vulguis). Els valors poden ser números enters o decimals.

### Funcionalitats

El programa ha de:

- Demanar el primer número
- Demanar el segon número
- Preguntar quina operació es vol fer (un dels següents símbols):
  - `+` suma
  - `-` resta
  - `*` multiplicació
  - `/` divisió
  - `%` rest de la divisió
- Mostrar el resultat de l'operació

---

## Funcionament

- El programa s’ha d’executar en bucle (loop) de manera contínua.
- Només s’aturarà quan l’usuari introdueixi una operació no vàlida (per exemple: `$`).

---

## Requisits del codi

- Utilitzar **variables en camelCase**
- Afegir **comentaris** per explicar el funcionament del codi
